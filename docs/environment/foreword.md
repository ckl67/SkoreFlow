# Foreword

[← back](./../index.md)

SkoreFlow is an open-source platform dedicated to the processing and management of sheet music.

Starting from a simple upload, the application transforms documents into structured musical data through a modern REST architecture designed for scalability, maintainability, and future extensions.

While the original inspiration came from [SheetAble](hhttps://github.com/ScoreAble/SheetAble) project, SkoreFlow has been entirely redesigned from the ground up with its own architecture, implementation, development workflow, and feature set.

Today, it stands as an **independent project focused on clean architecture, extensibility, and modern software engineering practices**.
